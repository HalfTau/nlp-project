If you have save game errors because of games for windows live, then uninstall the old gfwl that the game automatically installs and install all 3 games for windows live parts manually and restart the computer. To download the parts, type in google gfwlclient first link, wllogin64 second link and xliveredist first link.The game needs games for windows live profile to save the game progress. In the create profile window, scroll down and create a local profile.
